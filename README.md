Video Game Sales 电子游戏销售分析<br>
* 数据集：Video Game Sales<br>
* 该数据集包含游戏名称、类型、发行时间、发布者以及在全球各地的销售额数据。<br>
* 数据量：11列共1.66W数据。
* 基于这个数据集，可进行以下问题的探索：
* 1、电子游戏市场分析，包括受欢迎的游戏、类型、发布平台、发行人等；
* 2、预测每年电子游戏销售额；
* 3、可视化展示这个销售故事。
